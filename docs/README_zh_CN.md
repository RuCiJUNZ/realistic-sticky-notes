## 功能亮点

### 拟真便利贴
- **零语法 :** 双击即可新建或编辑内容，可以复制粘贴便利贴！
- **自定义 :** 支持切换 Style、Size、Pin Type（胶带/图钉/回形针）、background（支持图片png，jpg）、color、Shape。
- **markdown兼容:** 完美支持 Obsidian 自带的 Markdown 语法，支持直接粘贴图片（支持gif）
- **跨平台体验:** 兼容 iOS 和 Android 移动端。

### 画布管理
- **视图:** 提供“全览视图”与“居中视图”，支持一键“摆正”或“随意旋转”。
- **画布:** 预设 6 种背景画布，支持新建或删除多个独立画布，可以调整画布高度！
- **栏宽:** 顶部工具栏支持“缩减栏宽”，为你腾出更多的视觉空间（仅对当前便利贴md文件做出修改，不会影响全局设置）。


### 数据存储
- **安全:** 每一张便利贴的数据都以独立的 `.md` 文件存储在你的仓库中。

---

## 使用方法

1. 在任意 `.md` 文件中进入**编辑模式**。

2. 输入命令即可插入便利贴面板。

3. **双击**空白处：新建便利贴。

4. **右键**点击：调整外观、复制或删除。
